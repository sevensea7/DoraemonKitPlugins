## Installation

DoraemonKitPlugins is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
source 'https://github.com/sevensea7/Specs.git'
pod 'DoraemonKitPlugins'
```
